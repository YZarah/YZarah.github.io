markdown基本语法

发表于 2016-09-23   |   分类于 markdown   |  

Markdown 语法的目标是：成为一种适用于网络的书写语言。
Markdown 的语法种类很少，只对应 HTML 标记的一小部分。Markdown 的理念是，能让文档更容易读、写和随意改，Markdown 的格式语法只涵盖纯文本可以涵盖的范围。
下面介绍一下markdown经常使用到的语法。

标题

标题主要有两种写法，一种通过#来实现：通过“#”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）,另一种则是通过===和—实现：他们分别表示一级标题和二级标题。

实现方法如下：

### 三级标题
#### 四级标题

另一种实现：

一级标题
======
二级标题
-------

引用

在我们写作的时候经常需要引用他人的文字，这个时候引用这个格式就很有必要了，在 Markdown 中，你只需要在你希望引用的文字前面加上 > 就好了，例如：

>一级引用
>>二级引用

实现效果如下：

一级引用

二级引用

代码

需要引用代码时，如果引用的语句只有一段，不分行，可以用`将语句包起来。如果引用的语句为多行，可以将```置于这段代码的首行和末行。多行的代码也可以通过一个tab+4个空格来实现。

粗体与斜体

Markdown 的粗体和斜体也非常简单， 用两个 或 _ 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。例如：
粗体1 粗体2
斜体1
